# vue-tsx-calendar

#### Calendar app builded with Vue.js, JSX and Typescript

[Deployed app](https://chervonnyy.github.io/vue-tsx-calendar/)

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```

### Linting

```
yarn lint
```
